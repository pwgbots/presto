Verify letter of acknowledgment

Project relay acknowledge­ment letters are issued as PDF documents that feature a unique 32-digit authentication code in the upper left corner.

You can verify the authenticity of the letter either by clicking on this code, or by copying it to the input field below. If the code is valid, the details on the letter will be shown, allowing you to verify whether they match with the document text.

Verify badge

Project relay badges are issued as PNG images that feature a circle with a runner icon similar to the image on the left. The image itself contains invisible markings that encrypt infor­mation on the achievement represented by the badge.

You can verify the authenticity of a badge by first saving it as a PNG file and then loading this file into the input field below. If the badge is valid, the information it contains on will be shown.
